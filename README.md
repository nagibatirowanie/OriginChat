# OriginChat

## Описание
OriginChat - это продвинутый плагин для форматирования чата в Minecraft с поддержкой модульной системы. Плагин позволяет настраивать формат сообщений в чате, использовать HEX цвета и MiniMessage для стилизации текста, а также легко расширять функциональность с помощью модулей.

## Возможности
- Модульная система для легкого расширения функциональности
- Поддержка HEX цветов в сообщениях
- Поддержка MiniMessage для продвинутого форматирования текста
- Автоматическое обновление конфигурационных файлов
- Система управления модулями через команды

## Требования
- Paper 1.17-1.21.5
- Java 16 или выше

## Установка
1. Скачайте последнюю версию плагина из раздела Releases
2. Поместите JAR-файл в папку plugins вашего сервера
3. Перезапустите сервер или используйте плагин для загрузки плагинов
4. Настройте конфигурационные файлы по вашему усмотрению

## Конфигурация
После первого запуска плагин создаст следующие конфигурационные файлы:

### config.yml
Основной конфигурационный файл плагина, содержащий общие настройки и список включенных модулей.

```yaml
# Основной конфиг плагина OriginChat
version: 1

# Общие настройки плагина
settings:
  # Включить/выключить отладочные сообщения
  debug: false
  # Префикс плагина в сообщениях
  prefix: '&7[&bOriginChat&7]'

# Настройки форматирования чата
chat:
  # Формат сообщений в чате (поддерживает MiniMessage и HEX цвета)
  format: '<gray>[<color:#00AAFF>%player_name%</color>]</gray> <white>%message%</white>'
  # Включить/выключить поддержку HEX цветов
  hex-colors: true
  # Включить/выключить поддержку MiniMessage
  mini-message: true
  # Максимальная длина сообщения
  max-message-length: 256

# Настройки модулей
modules:
  # Включенные модули
  enabled:
    - 'chat'
```

### modules/chat.yml
Конфигурационный файл модуля чата, содержащий настройки форматирования сообщений и фильтрации.

## Команды
- `/originchat help` - Показать справку по командам
- `/originchat reload` - Перезагрузить плагин
- `/originchat module list` - Показать список доступных модулей
- `/originchat module enable <имя>` - Включить модуль
- `/originchat module disable <имя>` - Выключить модуль

## Права
- `originchat.admin` - Доступ к административным командам плагина
- `originchat.module.chat` - Доступ к функциям модуля чата

## Разработка модулей
Для создания собственного модуля необходимо:

1. Создать класс, наследующийся от `AbstractModule` или реализующий интерфейс `Module`
2. Реализовать методы `onEnable()` и `onDisable()`
3. Зарегистрировать модуль в `ModuleManager`

Пример простого модуля:

```java
public class MyModule extends AbstractModule {

    public MyModule(OriginChat plugin) {
        super(plugin, "mymodule", "Мой модуль", "Описание модуля", "1.0");
    }

    @Override
    public void onEnable() {
        // Код инициализации модуля
        log("Модуль включен!");
    }

    @Override
    public void onDisable() {
        // Код выключения модуля
        log("Модуль выключен!");
    }
}
```
